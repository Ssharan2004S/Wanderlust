🌍 WanderLust

A cozy Airbnb-style stay booking platform.

WanderLust is a full-stack project inspired by Airbnb, built to explore how real-world rental and booking systems work. It lets users browse unique stays, create listings, make reservations, and manage their trips—all through a clean, modern UI.

✨ Features

🏡 Browse Listings — View stays with images, descriptions, and pricing.

✍️ Host Your Property — Create, edit, and delete your own listings.

📅 Book & Manage Trips — Reserve listings and handle your bookings.

🔐 User Authentication — Secure login and signup flow.

📱 Fully Responsive — Works smoothly across phones, tablets, and desktops.
